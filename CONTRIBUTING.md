# Contributing

## Building locally

Open the project in Android Studio and run it on a device or emulator running Android 8.0+. No API keys or external services are required.

## Reporting a bug

Open an issue using the [bug report template](https://github.com/adiamas/umpireassistant/issues/new?template=bug_report.md). Include your app version (found in Settings → About), Android version, and steps to reproduce.

## Suggesting a feature

Open an issue describing the feature and the problem it solves. Please open an issue before submitting a pull request so we can discuss the approach first.

## Submitting a pull request

1. Fork the repository and create a branch from `main`
2. Make your changes
3. Open a pull request with a clear description of what changed and why
